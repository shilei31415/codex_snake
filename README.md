# 小游戏合集

一个纯前端单页小游戏合集，目前包含贪吃蛇和连连看。项目不需要构建工具，直接用浏览器打开 `index.html` 即可游玩。

## 游戏列表

- 贪吃蛇：使用方向键或 WASD 控制移动，空格暂停。吃到食物会变长，撞墙或撞到自己游戏结束。
- 连连看：点击两张相同麻将牌，如果路径不超过两次转弯，就可以连线并消除。

## 本地运行

方式一：直接双击打开 `index.html`。

方式二：启动一个本地静态服务。

```bash
python -m http.server 5173
```

然后在浏览器访问：

```text
http://127.0.0.1:5173/index.html
```
